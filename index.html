AWB Köln Abfuhrkalender ICS Data Generator

GitHub Repository

Die Daten werden nur an AWB Köln direkt gesendet, es werden keine Daten von dieser Webseite gespeichert

Nach Eingabe der Daten den Inhalt der Textbox mit einem Texteditor als *.ics speichern & in den Kalender der Wahl importieren (oder einfach "Download" klicken, wenn es der Browser unterstützt)
